# Python_turtle_game
### 주요 내용
1. 토끼 , 거북이 가위바위보 대결 게임
2. 토끼는 컴퓨터, 거북이는 게임 이용자
3. 토끼는 랜덤으로 '가위', '바위' , '보' 중에 하나를 고르고 거북이는 아래 '가위' , '바위' , '보' 중 버튼 하나를 골라 클릭
4. 이기면 앞으로 가고 지면 뒤로 비기면 둘다 앞으로 조금 감
5. 먼저 결승 선에 닿으면 이김
